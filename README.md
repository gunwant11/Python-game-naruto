# PygameForBeginners
A simple 2D python game designed.

[Download](https://github.com/gunwant11/Python-game-naruto/releases/tag/v1.0)


Controls

| Keys        | Naruto      | Sasuke        |
| :---        |    :----:   |          ---: |
| Shoot       | ctrl left   | ctrl right    |
| up          | w           | up key        |
| down        | s           | down key      |
| left        | d           | left key      |
| right       | a           | right key     |
